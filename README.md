# Calculadora em Python

## 📌 Objetivo do Projeto

Este projeto foi desenvolvido em Python com o objetivo de praticar os conceitos básicos da programação, criando uma calculadora interativa executada pelo terminal.

A aplicação permite ao usuário realizar operações matemáticas fundamentais, escolhendo a operação desejada através de um menu e informando os valores a serem calculados.

## 🎯 Funcionalidades

* Adição de números.
* Subtração de números.
* Multiplicação de números.
* Divisão de números.
* Menu interativo no terminal.
* Execução contínua até que o usuário escolha encerrar o programa.

## 📚 Aprendizados

Durante o desenvolvimento deste projeto foram praticados os seguintes conceitos:

* Programação em Python.
* Entrada e saída de dados (`input` e `print`).
* Conversão de tipos (`int` e `float`).
* Estruturas condicionais (`if`, `elif` e `else`).
* Estruturas de repetição (`while`).
* Operadores matemáticos.
* Organização de menus interativos.
* Limpeza da tela utilizando a biblioteca `os`.
